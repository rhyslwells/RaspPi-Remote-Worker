# Setting up the Raspberry pi from Scratch

1) Use Raspberry Pi Imager (https://www.raspberrypi.com/software/)
to flash the SD card with Raspberry Pi OS (64-bit)
   1) You get the raspberry pi log in details here, username and password.
   2) This will also connect to the wifi
2) find raspberry pi IP address. Use IP address and search in chrome browser to see network devices. Look for raspberry pi and note the IP address. 
   - Open browser: `http://192.168.1.1` (or whatever your gateway is)
   - Username/Password: Usually on your router (e.g., admin/admin or admin/password)
3) Use ssh to connect to the raspberry pi from your terminal: `ssh username@IP_ADDRESS`
4) use raspberry pi connect to connect to the raspberry pi from your browser: https://connect.raspberrypi.com/devices
